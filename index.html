Hydrologic Database (HDB) Data Query

The Bureau of Reclamation makes efforts to maintain the accuracy of data found in the HDB system databases but the data is largely unverified and should be considered preliminary and subject to change. Select your desired database:
LCHDB    UCHDB    YAOHDB    ECAOHDB   LBOHDB
Input your desired Site-Datatype ID number(s) or select from the drop-down list:
Separate multiple ID numbers with commas and no spaces. (Ex: 1234,5678,...)
Select your desired data frequency:
Instantaneous   Hourly   Daily   Monthly   Yearly
Select your desired date range (MM-DD-YYYY):
From To
Select your desired data type:
Observed (R-Tables)   Modeled (M-Tables), MRID =
Select your desired output format:
CSV   HTML Table   JSON   Interactive Plot

You may copy and save the URL that is generated by the query once you press submit. This URL will allow you to access the query again at a later time.

Web Service Information

Sample URL Query Pattern: http://ibr3lcrsrv01.bor.doi.net:8080/HDB_CGI.com?svr=lchdb2&sdi=1863,1930,2166,2146&tstp=DY&t1=1-1-1980&t2=1-1-2016&format=9
svr - either lchdb2, uchdb2, or yaohdb for the LC, UC, or YAO HDBs respectively
sdi - use a comma separated string for multiple sdis.
tstp - IN, HR, DY, MN, YR to query instantaneous, hourly, daily, monthly, or yearly data in the appropriate HDB table
t1 - a negative number (-X) to grab the last X time-steps of data
OR
t1 & t2 - start and end date in m/d/yyyy format
format - 1, 2, 3, 4, 9, table, csv, or chart to display the data differently. chart is for an interactive chart.
OPTIONAL
table=M&mrid=XXX& - you can add this before 'format=X' to query data from the M-Tables in HDB. This query does not work with the interactive chart.
